He aqui una pequeña aplicación que muestra el funcinamiento de spring security de forma basica y utilizando JWT para proteger las apis, asi que son dos aplicaciones:

La aplicación que envuelve todo el Fronted

La aplicación que contiene las APIs, protegidas por JWT

Esta aplicación esta diseñada para simular el comportamiento Angular o un sistema tipo one page. Y ha sido contruida con Javascript y CSS.

Muestra diversas funcionalides para tomarlos de ejemplo en aplicaciones futuras.
